# MCP Server for PDF Tools

## Overview
This project provides an MCP server with tools for PDF manipulation, including combining, splitting, compressing, and converting PDFs.

## Directory Structure
- `tools/`: Contains individual tool implementations.
- `server/`: Core MCP server logic.
- `config/`: Configuration files.
- `tests/`: Unit tests for tools.

## Quick Start
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the server:
   ```bash
   python server/mcp_server.py
   ```
