# Foxit PDF File Upload Widget

A file upload widget based on Vue 3 + Vite + Arco Design.

## Features

- ✨ Drag and drop upload support
- 📦 Batch upload support
- 📝 File list display
- 🗑️ File deletion
- ➕ Add more files
- 💅 Styled with Arco Design UI

## Install Dependencies

```bash
yarn install
```

## Development

```bash
yarn dev
```

## Build

```bash
yarn build
```

## Preview

```bash
yarn preview
```

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Arco Design Vue
